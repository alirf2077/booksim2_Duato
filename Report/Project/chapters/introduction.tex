

\فصل{مقدمه}

شبیه‌ساز \lr{Booskim} یکی از مهم‌ترین شبیه‌سازهای حضوزه شبکه‌های میان‌ارتباطی است که سالیان طولانی است در این حوزه مورد استفاده قرار گرفته و در مقالات و پروژه‌های زیادی به منظور شبیه‌سازی ایده‌های نوین این حوزه مورد استفاده قرار گرفته است. این شبیه‌ساز براساس مطالب کتاب 
\lr{Principles and Practices of Interconnection Networks}
ساخته شده است.

در این پروژه قصد داریم به شبیه‌سازی یک
 \lr{k-ary n-cube}
یا همان شبکه توری مدور (Torus) پرداخته و الگوریتم مسیریابی \lr{Duato} را با حداکثر تعداد کانال‌های ممکن برای مسیریابی
\lr{ Fully Adaptive }
 و حداقل تعداد ممکن برای مسیریابی
\lr{  Deadlock Free}
   پیاده‌سازی کنیم.


منابع مورد استفاده برای این گزارش \cite{dally2004principles} و \cite{jiang2010booksim} بوده‌اند.
